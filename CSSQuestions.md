### 1. center an element vertically

```css
.outer {
  width: 100%;
  height: 100%;
  background-color: #efefef;
  /* your code here */
  display: flex;
}

.inner {
  width: 100px;
  height: 100px;
  background-color: #f44336;
  /* your code here */
  margin: auto;
}
```

### 2. truncate text in one line(with ellipsis)

```css
.one-line {
  /* your code here */
  overflow: hidden;
  white-space: nowrap;
  text-overflow: ellipsis;
}
```

### 3. truncate text in multiple lines(with ellipsis)

```css
.max-three-lines {
  /* your code here */
  display: -webkit-box;
  overflow: hidden;
  -webkit-line-clamp: 3;
  -webkit-box-orient: vertical;
}
```

### 4. two-column layout

```css
.container {
  height: 300px;
  /* your code here */
  display: flex;
}

.left {
  background-color: #f44336;
  /* your code here */
  min-width: 100px;
  flex: 1;
}

.right {
  background-color: #2973af;
  /* your code here */
  flex: 3;
}
```
